# USD.AI Adventure - Interactive Learning

An interactive, gamified educational website that teaches USD.AI concepts in a fun, easy-to-understand way.

## Features

- **Level-based learning** - Progress through 5 levels from basics to expert
- **Interactive demonstrations** - Click to mint USDai, stake for sUSDai, and see yield grow
- **Achievement system** - Unlock badges as you learn
- **XP tracking** - Earn points for completing sections
- **Yield calculator** - See how your money could grow
- **Final quiz** - Test your knowledge with 5 questions

## Concepts Covered

1. **USDai** - A stablecoin worth $1, always redeemable
2. **sUSDai** - Staked USDai that earns 10-15% APR
3. **GPU Financing** - How yield comes from AI infrastructure
4. **Three Participants** - Depositors, Borrowers, and Curators
5. **The Flow** - How money moves through the system

## Running Locally

```bash
# Navigate to the project
cd usdai-learn

# Start a local server
python3 -m http.server 8080

# Open in browser
open http://localhost:8080
```
